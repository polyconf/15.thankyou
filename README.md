# kulfon

Static site generator that uses [hugo][1] and [broccoli][2]

## Install

    brew install go
    brew install hugo

    git clone http://github.com/nukomeet/kulfon YOUR_NAME

    cd YOUR_NAME

    npm install


## Usage

    cd YOUR_NAME
    npm start


[1]: http://gohugo.io/
[2]: https://github.com/broccolijs/broccoli
